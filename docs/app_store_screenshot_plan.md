# App Store Screenshot Plan

TalkLogのApp Store / TestFlight向けスクリーンショット準備メモです。

Apple公式のApp Store Connectヘルプでは、スクリーンショットはアプリの画面を表示し、各デバイス種別ごとに最大10枚までアップロードできます。初回リリースでは、訴求が散らばらないようにiPhone縦向き5枚を基本セットにします。

## 基本方針

- 最初はiPhone向け5枚で作る
- 画面は実機またはエミュレータで撮影する
- 個人情報や実在メールアドレスは写さない
- テスト用アカウント、テスト用録音、テスト用添削結果を使う
- スクショ内の文章は短く、機能説明より利用価値を優先する
- 課金未実装のため、Premium登録完了を示す画面は撮らない

## 推奨スクショ構成

| No | 画面 | 目的 | 上に載せる訴求文 | 撮影前の状態 |
|---:|---|---|---|---|
| 1 | ホーム / 進捗 | アプリの価値を最初に伝える | 自分の声で、成長を記録 | 録音数、連続日数、容量メーターが見える状態 |
| 2 | 録音画面 | メイン機能を伝える | 話した英語をすぐ録音 | 録音ボタン、言いたいこと入力、読み上げメモが見える状態 |
| 3 | 録音履歴 | 継続利用の価値を伝える | 1週間前の自分と比べられる | 複数の録音履歴が新しい順で表示される状態 |
| 4 | AI添削結果 | AI価値を伝える | 文字起こしと添削で振り返る | 添削済みの録音詳細を表示 |
| 5 | 単語ランキング / 単語帳 | 学習の積み上げを伝える | よく使う単語を見える化 | 単語ランキングとアドバイスが見える状態 |

## 追加候補

5枚で足りない場合、以下を追加します。

| No | 画面 | 訴求文 |
|---:|---|---|
| 6 | Premium案内 | 制限を気にせず学習を続ける |
| 7 | 設定 / 言語変更 | 学習言語ごとに記録を整理 |
| 8 | 単語帳 | 気になった表現を復習 |
| 9 | 容量メーター | 音声ログを安心して管理 |
| 10 | 録音補助 | 何を話すか迷っても大丈夫 |

## 撮影用デモデータ

スクショ前に以下のデータを用意します。

### アカウント

```text
Email: screenshot-test@example.com
Role: TESTER
Language: スペイン語 または 英語
```

実際のメールアドレスを写さないように、撮影用アカウントを使います。

### 録音履歴

履歴には3〜5件あると見栄えが良いです。

例:

| 表示タイトル | 言語 | 内容 |
|---|---|---|
| 朝の練習 | 英語 | 今日の予定を話す |
| 仕事終わりの練習 | 英語 | 疲れたが継続したい |
| 週末の予定 | 英語 | 友人と会う予定 |
| 旅行の練習 | スペイン語 | ホテルで使う表現 |

### AI添削結果サンプル

添削画面に見せたい要素:

- 文字起こし
- 自然な表現
- 日本語訳
- 文法メモ
- 語彙メモ
- スコア
- 励ましコメント

### 単語ランキングサンプル

表示したい単語例:

| 単語 | 言い換えアドバイス |
|---|---|
| tired | exhausted, worn out も使えます |
| work | job, task, project と言い換えられます |
| practice | train, work on と表現できます |
| want | would like to にすると丁寧です |
| today | this morning, this evening で具体化できます |

## 撮影前チェックリスト

- [ ] 本番・個人メールアドレスが画面に出ていない
- [ ] APIキー、Supabase URL、内部IDが画面に出ていない
- [ ] 赤いエラー文が残っていない
- [ ] 録音履歴の日付が日本時間で自然
- [ ] FREE上限エラーなどネガティブな画面を撮っていない
- [ ] Premium登録完了のような未実装表現を出していない
- [ ] ボタンや文字が切れていない
- [ ] ステータスバーの時刻・電池表示が不自然でない
- [ ] スクショ内の言語が日本語で統一されている
- [ ] 音声や個人情報を含む実データではなくデモデータを使っている

## ファイル命名

撮影した画像は、以下のように保存します。

```text
screenshots/ios/01-home-progress.png
screenshots/ios/02-recording-helper.png
screenshots/ios/03-history.png
screenshots/ios/04-ai-feedback.png
screenshots/ios/05-word-ranking.png
```

```text

screenshots/ios/07-language-settings.png
screenshots/ios/08-vocabulary.png

```

## App Store Connect入力時のメモ

- 最初の1枚はアプリの価値が一目で分かる画面にする
- 同じような画面を並べすぎない
- iPhone用のスクショから先に用意する
- iPad対応を有効にする場合はiPad用スクショも必要になる可能性がある
- iPadを積極対応しない場合、iPhone中心の表示崩れ確認を優先する
