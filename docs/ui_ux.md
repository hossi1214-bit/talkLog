\# UI\_UX.md



\# TalkLog UI / UX設計書



\## 目的



本ドキュメントは、TalkLogのUIデザイン・UX・画面遷移・デザインルールを定義する。



TalkLogは「語学学習アプリ」よりも、「毎日続けられるライフログアプリ」に近い体験を提供することを目指す。

\# UX方針



ユーザーは「学習する」のではなく、「話すだけ」で良い。



TalkLogが自動で



\- 記録

\- 添削

\- 分析

\- 振り返り



を行う。



ユーザーの操作回数は最小限にする。



\---



\# 長期ビジョン



TalkLogは、語学学習アプリというよりも「毎日のスピーキングを自然に習慣化できるAIジャーナル」を目指す。



UI/UXは、継続しやすさ・安心感・達成感を最優先に設計する。
