## Beta App Description

```text
TalkLogは、英語スピーキング練習を録音し、AI添削や学習履歴として活用する音声ログアプリです。

このベータ版では、録音、再生、履歴保存、AI添削、単語帳、単語ランキング、学習進捗、Premium案内画面などの基本機能を確認できます。
```

## Review Notes for Apple

App Store / TestFlight審査メモに貼る文言です。

```text
TalkLogは、ユーザーがスピーキング練習を録音し、学習履歴として保存できる語学学習アプリです。

マイク権限は録音機能のために使用します。録音した音声はユーザー自身の学習履歴として保存され、アプリ内で再生・削除できます。

AI添削機能では、録音音声をSupabase Edge Functions経由でOpenAI APIに送信し、文字起こしと添削結果を取得します。

現在のPremium登録ボタンはプラン案内画面と比較表の表示用です。App Store課金処理はまだ有効化していないため、テスト中に実際の購入は発生しません。
```

## Test Account Template

```text
Email: screenshot-test@example.com
Password: REVIEW_TEST_PASSWORD
```

テストアカウント作成後、Supabase SQL Editorで必要に応じて権限を付与します。

```sql
update public.profiles p
set role = 'TESTER'
from auth.users u
where p.id = u.id
  and u.email = 'screenshot-test@example.com';
```

## Tester Account Operation

初期運用では、テスター自身にアプリでメールアカウント登録してもらい、登録メールアドレスを受け取ってから管理者がSupabaseで `TESTER` を付与します。

| 手順 | 担当 | 内容 |
|---:|---|---|
| 1 | テスター | アプリでメールアドレス・パスワード登録 |
| 2 | テスター | 登録したメールアドレスを管理者へ連絡 |
| 3 | 管理者 | Supabaseで `profiles.role` を `TESTER` に変更 |
| 4 | テスター | アプリを再起動、またはログアウトして再ログイン |
| 5 | テスター | TESTER権限で有料機能相当を確認 |
