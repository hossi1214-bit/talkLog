# TestFlight Review Notes Draft

TestFlight審査、内部テスター、外部テスター向けの説明下書きです。

## Beta App Description

```text
TalkLogは、英語スピーキング練習を録音し、AI添削や学習履歴として活用する音声ログアプリです。

このベータ版では、録音、再生、履歴保存、AI添削、単語帳、単語ランキング、学習進捗、Premium案内画面などの基本機能を確認できます。
```

## What to Test

```text
以下の流れを中心に確認してください。

1. メールアカウントで登録・ログインする
2. 録音画面でスピーキング練習を録音する
3. 録音を再生できることを確認する
4. 録音履歴が新しい順に表示されることを確認する
5. 録音詳細からAI添削を実行する
6. 文字起こし、添削、自然な表現、日本語訳が表示されることを確認する
7. 単語帳に単語を追加し、復習済みチェックを付ける
8. よく使う単語ランキングと言い換えアドバイスを確認する
9. 設定画面で学習言語を変更する
10. Premium案内画面と比較表を確認する
```

## Review Notes for Apple

```text
TalkLogは、ユーザーがスピーキング練習を録音し、学習履歴として保存できる語学学習アプリです。

マイク権限は録音機能のために使用します。録音した音声はユーザー自身の学習履歴として保存され、アプリ内で再生・削除できます。

AI添削機能では、録音音声をSupabase Edge Functions経由でOpenAI APIに送信し、文字起こしと添削結果を取得します。

現在のPremium登録ボタンは案内画面と比較表の表示用です。App Store課金処理はまだ有効化していないため、テスト中に実際の購入は発生しません。
```

## Internal Tester Checklist

| No | 確認項目 | 期待結果 |
|---:|---|---|
| 1 | 新規登録 | メールで登録できる |
| 2 | ログイン維持 | アプリ再起動後もログイン状態が残る |
| 3 | 録音 | 録音開始、停止、保存ができる |
| 4 | 録音キャンセル | キャンセル時は履歴に保存されない |
| 5 | 録音補助 | 日本語入力から練習文を作成できる |
| 6 | 履歴 | 新しい録音が上に表示される |
| 7 | 再生 | 履歴から音声を再生できる |
| 8 | AI添削 | 文字起こしと添削が表示される |
| 9 | FREE制限 | 1日5回を超えると制限メッセージが出る |
| 10 | TESTER権限 | TESTERは有料機能を利用できる |
| 11 | 単語帳 | 添削結果から単語帳へ追加できる |
| 12 | ランキング | 文字起こし由来の単語ランキングが表示される |
| 13 | 容量メーター | Supabase保存容量が表示される |
| 14 | ログアウト | ログアウト後はユーザーデータが見えない |
